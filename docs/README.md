# 🌐 LiseryMenu - Сайт документации

Красивый одностраничный сайт с полной документацией по плагину LiseryMenu.

## 📁 Файлы

- `index.html` - Главная страница сайта
- `README.md` - Этот файл

## 🚀 Запуск

### Локально
Просто откройте `index.html` в браузере!

### На хостинге
1. Загрузите файлы на хостинг (GitHub Pages, Netlify, Vercel и т.д.)
2. Готово!

## ✨ Возможности сайта

- ✅ Современный адаптивный дизайн
- ✅ Плавная прокрутка
- ✅ Подсветка активных секций
- ✅ Красивые карточки и таблицы
- ✅ Примеры кода с подсветкой
- ✅ Адаптация под мобильные устройства

## 📱 Секции сайта

1. **Hero** - Главный баннер с кнопками
2. **Features** - Ключевые возможности (6 карточек)
3. **Installation** - Инструкция по установке
4. **Documentation** - Полная документация
5. **Examples** - Примеры конфигураций
6. **Download** - Скачивание плагина
7. **Comparison** - Сравнение с DeluxMenus
8. **Footer** - Футер с ссылками

## 📝 Как редактировать

### Добавить новую секцию
```html
<div class="section" id="новая-секция">
    <h2>🎯 Заголовок</h2>
    <p>Содержимое...</p>
</div>
```

### Добавить карточку
```html
<div class="card">
    <h4>Заголовок</h4>
    <p>Текст</p>
</div>
```

### Добавить таблицу
```html
<table>
    <thead>
        <tr>
            <th>Колонка 1</th>
            <th>Колонка 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Данные 1</td>
            <td>Данные 2</td>
        </tr>
    </tbody>
</table>
```

### Добавить алерт
```html
<div class="alert alert-success">
    <strong>✅ Успех!</strong> Сообщение
</div>
```
